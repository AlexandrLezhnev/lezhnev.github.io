The first version of the program (aka Adobe Premiere 1) was released on August 21, 2003 for Windows operating systems. Since the third version, the program has become available for OS X operating systems. The first two versions were released as separate products, the third version was released as part of the Adobe Creative Suite 3 package. Starting from the fifth version, only 64-bit operating systems are supported, while the fourth version was also intended to be used in 32-bit ones. Premiere Pro is used by companies such as The BBC, The Tonight Show. It was used for editing many films, in particular such as "The Social Network", "Through the Dust to Victory", "Captain Abu Raed", Deadpool.
